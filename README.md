## Shortcut

- `Command + Left`: Go back
- `Command + Right`: Go Forward
- `Command + r`: Go to homepage
- `COmmand + o`: Open current page externally

## Build

```bash
$ yarn

$ npm run app # to start the app

$ npm run dist # to build the app distribution
```
